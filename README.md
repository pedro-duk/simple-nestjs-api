## Informações gerais
* <b> Objetivo: </b> criação de um repositório de uma API Simples usando diversas funcionalidades do Nest.JS

## Funcionalidades
* Criação de Validadores personalizados com @ValidatorConstraint



## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
